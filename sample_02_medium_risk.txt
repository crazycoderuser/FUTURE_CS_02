----- ANALYST NOTES -----

RISK LEVEL : SUSPICIOUS (MEDIUM)
SCORE      : 6 points
INDICATORS : 4 detected

[HIGH]   Sender domain — amazon-support-help.org is not amazon.com.
         This is subdomain/domain spoofing of the Amazon brand.
[MEDIUM] SPF: SOFTFAIL — message may be legitimate but server configuration
         is incorrect, or sender is unauthorized.
[MEDIUM] Generic greeting — "Dear Customer" rather than registered name.
[LOW]    Vague reason — "verification issue" with no order number cited.

NOTE: DKIM passes for this domain, which makes it appear more legitimate.
However, DKIM passing for a spoofed domain means only that the fake domain
signed the message — not that the sender is Amazon.

RECOMMENDED ACTION: Do not click link. Verify order status by logging in
directly to amazon.com in a new browser tab. Report to IT if received at
a work address.
